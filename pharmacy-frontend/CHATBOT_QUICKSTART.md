# 🚀 Quick Start: Medicine Chatbot

## Get Started in 3 Steps (5 minutes!)

### 1️⃣ Get Your FREE API Key
Visit: **https://makersuite.google.com/app/apikey**
- Sign in with Google (no credit card!)
- Click "Create API key"
- Copy the key

### 2️⃣ Create `.env` File
```bash
# Create .env file in project root
echo "VITE_GEMINI_API_KEY=paste_your_key_here" > .env
```

Or manually create `.env` and add:
```
VITE_GEMINI_API_KEY=AIzaSyC...your_key_here
```

### 3️⃣ Restart Development Server
```bash
npm run dev
```

## ✅ Test It!
1. Login as a **customer**
2. Look for the **floating chat button** (bottom-right, blue/purple with 💊)
3. Click and ask: **"What is Paracetamol used for?"**

## 🎉 That's It!
You now have a working medicine chatbot that:
- ✅ Answers medicine questions only
- ✅ Provides uses, dosage, and side effects
- ✅ Completely FREE (1,500 requests/day)
- ✅ No backend required
